## ✨ Features

- **Basic operations** — addition, subtraction, multiplication, division
- **Percentage support** — `%` key for quick percentage calculations
- **Expression preview** — shows your full input above the result
- **Keyboard support** — use your keyboard to type numbers and operators
- **Error handling** — gracefully handles division errors and invalid input
- **Smooth animations** — card pop-in on load, button press feedback
- **Responsive display** — font shrinks automatically for long numbers

---

## ⌨️ Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `0–9` | Input digits |
| `+ - * /` | Operators |
| `.` | Decimal point |
| `%` | Percentage |
| `Enter` or `=` | Calculate |
| `Backspace` | Delete last character |
| `Escape` | Clear all |

---

## 🎨 Design

- **Style:** Glassmorphism with frosted blur and soft borders
- **Background:** Animated gradient mesh (lavender → sky blue)
- **Fonts:** [DM Mono](https://fonts.google.com/specimen/DM+Mono) for the display, [Nunito](https://fonts.google.com/specimen/Nunito) for buttons
- **Colors:** Purple-tinted operators, blue equals, rose clear button
- Loads Google Fonts via CDN — requires an internet connection for correct typography
